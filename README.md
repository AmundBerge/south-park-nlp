# Authorship attribution in South Park

For a project in TDT4310 at NTNU

The dataset used is "South Park Dialogue", available on [Kaggle](https://www.kaggle.com/datasets/tovarischsukhov/southparklines). 

For running the project:
- Download the dataset, name it "All-seasons.csv", and place it in data/ at the project root
- run "pip install -r requirements.txt" in your terminal. Note that you may need to run a [Python virtual environment](https://docs.python.org/3/library/venv.html#how-venvs-work), depending on your OS
- run the two notebook files in order
