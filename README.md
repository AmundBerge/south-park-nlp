# Authorship attribution in South Park

For a project in TDT4310 at NTNU

The dataset used is "South Park Dialogue", available on [Kaggle](https://www.kaggle.com/datasets/tovarischsukhov/southparklines). 

For running the project:
- Download the dataset, name is "All-seasons.csv" and place it in data/ at the project root
- run "pip install -r requirements.txt" in your terminal
- run the two notebook files in order
